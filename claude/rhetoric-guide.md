# English — Rhetorical Skills Strategy Guide
 
*At a 31 → 36 level, rhetoric questions are where the points are. These require judgment, not just rule knowledge.*
 
---
 
## The 5 Rhetoric Question Types
 
### 1. Transitions
**What it asks:** Which word or phrase best connects this sentence to the previous one?
 
**Strategy:**
- Read the sentence before AND the sentence in question
- Decide the logical relationship: Does it ADD (also, furthermore, in addition)? CONTRAST (however, although, on the other hand)? CONCLUDE (therefore, thus, as a result)? GIVE EXAMPLE (for example, for instance)?
- Never pick a transition just because it sounds smart — the logic must match
**Common wrong answer trap:** A transition that sounds sophisticated but reverses the meaning (using "however" when the two sentences actually agree)
 
---
 
### 2. Organization (sentence/paragraph placement)
**What it asks:** Where should this sentence/paragraph be placed? Should it be added or deleted?
 
**Strategy:**
- If placing a sentence: read all four options in context. The correct placement will have the sentence flow logically from the sentence before AND into the sentence after.
- If adding a sentence: ask "does this add relevant, specific information — or is it redundant/off-topic?"
- If deleting: "does removing this hurt the paragraph's meaning, or is it a digression?"
- The answer is almost always the option that maintains topic continuity
---
 
### 3. Strategy (purpose and effect)
**What it asks:** Does this addition accomplish the stated goal? Does this sentence serve a specific purpose?
 
**Strategy:**
- Read the question carefully — it tells you what to look for (e.g., "provides a specific example," "introduces a contrast," "emphasizes the main argument")
- Go back to the passage and ask: does this sentence literally do that thing?
- Yes/No questions: eliminate the wrong Y/N first, then pick the best reason
**Common trap:** An answer that correctly identifies what the sentence does but overstates it (e.g., "proves beyond doubt" when it just "suggests")
 
---
 
### 4. Style and Tone
**What it asks:** Which version is most consistent with the tone/style of the passage? Which is most concise?
 
**Strategy:**
- Identify the passage's register: formal/academic, conversational, narrative, journalistic
- Match that register. Don't add slang to a formal essay; don't add jargon to a casual one
- When two answers are grammatically correct, choose the more concise one
- Redundancy: if two words mean the same thing, one must go (*past history → history; end result → result*)
**Concision traps:**
- "The reason why is because" → "Because"
- "In spite of the fact that" → "Although"
- "At this point in time" → "Now"
- "Due to the fact that" → "Because"
---
 
### 5. Relevance (DELETE questions)
**What it asks:** Should this underlined sentence be kept or deleted?
 
**Strategy:**
- Read the paragraph without the sentence. Does it make complete sense? Does anything important disappear?
- "Delete it because it digresses from the main topic" is correct when the sentence introduces an unrelated idea
- "Keep it because it provides specific detail" is correct when the sentence adds concrete support for the paragraph's point
---
 
## The STOP Strategy for Rhetoric Questions
 
Before answering any rhetoric question:
 
**S — Summarize** what the sentence/passage is saying in one line  
**T — Tone** — what's the register of this passage?  
**O — Order** — what came before and what comes after?  
**P — Purpose** — what is this specific sentence trying to do?
 
---
 
## High-Value Patterns at the 35–36 Level
 
1. **The almost-right transition:** Two transitions both seem logical (e.g., "furthermore" vs "therefore") — read again for the precise logical shift
2. **The redundant addition:** An added sentence that repeats something already stated — always delete
3. **The off-topic digression:** Interesting but irrelevant detail — always delete
4. **The misplaced emphasis:** A sentence that is true but emphasizes the wrong aspect of the paragraph's argument
5. **The subtle tone mismatch:** An informal phrase in a formal passage, or vice versa
